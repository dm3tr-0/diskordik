📡 что это?

Diskordik — самоподнимаемый мессенджер для команды. Как Discord, но с сервером под вашим контролем.

Никаких ограничений. Не нужен „zapret/vpn“. Поднимаете на своём сервере — и пользуетесь.


  • Текстовые каналы
  • Голосовые каналы
  • Self-hosted — данные только ваши
  • Адаптивный веб‑интерфейс
  • Open source — можно допилить под себя

⚙️ репозиторий

Python 3 Flask SQLite WebRTC HTML5/CSS3

быстрый старт:

curl -sSL https://raw.githubusercontent.com/dm3tr-0/diskordik/main/install.sh | bash
